    • O que é HTML?

    A HTML - Linguagem de Marcação de Hipertexto é utilizada junto com CSS e Javascript para o desenvolvimento de sites. Seria inadequado dizer "Eu programo em HTML" visto que não funciona com instruções como as linguagens de programação. Focada no conteúdo do site, utiliza as chamadas tags para a construção do site.

    O que são as tags?

    É o conjunto de palavras entre os sinais de colchete angular (< >). As tags informam ao navegador a estrutura da página.

    • <!DOCTYPE html>: esta tag informa ao navegador a versão do HTML.

    A maioria das tags possuem abertura e fechamento, sendo o fechamento identificado pela barra. Exemplos:

    • <head> </head>: delimita o cabeçalho do documento;
    • <body> </body>: define o corpo do documento;
    • <h1> </h1> (de h1 até h6): define os títulos, sendo h1 o maior valor hierárquico decrescendo até h6 de menor valor. H1 utilizado para título principal e os outros como subtítulos;
    • <p> </p>: define o parágrafo
    • <ol> </ol> e <ul> </ul>: criam listas ordenadas e não ordenadas, respectivamente, utilizadas com <li> </li> que define um item da lista;
    • <sup> </sup>: define um texto sobrescrito - x3;
    • <sub> </sub> define um texto subscrito - H2O.

    As tags também podem ter atributos e valores:

    • <html lang="pt-br"> </html>: contém todos os elementos do documento HTML, com o atributo lang para definir o idioma como português brasileiro;
    • < a href="destino.html">Clique aqui </a>: tag a (âncora) utlizada para criar um link, com o atributo href que é a página destino.

    Algumas tags não possuem fechamento:

    • <br>: utilizada para quebrar linha;
    • <img>: utilizada para inserir uma imagem.
